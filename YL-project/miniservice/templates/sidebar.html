FAQ
How does this work?

This app uses flask running in Python to respond to requests.

What other libraries does it use?

This app uses scikit-learn's trees to predict your vote.

What dataset is it trained on?

This app uses ANES' 2020 election dataset. No filtering of any sort was done on the dataset.

What does this app use in the front end?

The browser side uses Bootstrap.

Do you store any of my data?

No, your data is deleted after your request is done (usually in less than 1 second).
